---

## 1. 页面定位

> **Settings 给信任，不给杂项堆叠。**

Settings 不能再像杂项仓库。在新版里，它承担的其实是"信任建立"职责。

## 3. 分类职责

| 分类 | 职责 |
|------|------|
| AI 配置 | 解释和配置本地 AI 分析能力，包括密钥、模型、连接状态、最近分析 |
| Sources 管理 | 配置源的行为和质量，包括 Starter Pack、健康状态、同步策略 |
| 外观阅读 | 调整阅读体验，包括主题、字号、行高、阅读器偏好、卡片密度 |
| 应用行为 | 控制系统级行为，包括开机启动、后台同步、通知、缓存和数据保留 |

详细字段、校验、保存策略、错误处理和验收清单见 `prd/ui/settings-interaction-spec.md`。

---

## 4. 对应版本任务

Settings 的重构分散在多个版本中：

- v2.1：AI 配置基础（API Key / Model）
- v2.7+：Sources 管理完善
- v2.x 后期：外观与阅读偏好完善
