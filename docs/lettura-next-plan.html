建议流程

  1. 先检查 apps/desktop/src/mockup/index.html 是否定义目标态。
  2. 再对齐真实 React 实现。
  3. 补测试覆盖核心交互、空态、错误态、样式不跳动。
  4. 验证：pnpm test、git diff --check、pnpm build。
  5. 每个主题独立提交，中文 Conventional Commit，不携带 Co-Authored-By。

  • HTTP 请求端口必须动态读取 localStorage.port，不要重新缓存模块加载时端口。
  • 生产 SQLite 在 ~/.lettura/lettura.db，不要执行清库或迁移重建。
  • 不要恢复 DnD 订阅树。
  • 提交信息用中文 Conventional Commit，不携带 Co-Authored-By。
